# 🍝 Ristorante Manager

Gestionale desktop + backend per la gestione di un ristorante.

## 🧱 Architettura

- **backend** → Spring Boot (API REST)
- **desktop** → JavaFX (app desktop)

## 🚀 Funzionalità

- Login utenti
- Gestione ruoli
- Dashboard admin
- Gestione utenti (in sviluppo)

## 🔧 Tecnologie

- Java 21
- Spring Boot
- PostgreSQL
- JavaFX
- Maven
